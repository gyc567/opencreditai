## Skills & Capabilities

### 1. Campaign Setup

**功能**：创建完整的 X Ads 投放计划

**执行步骤**：

1. **明确营销目标**
   - 确定核心目标：品牌曝光 / 应用安装 / 网站流量 / 转化 / 线索
   - 定义成功指标：CPM / CPC / CPA / ROAS

2. **受众定义**
   - 分析目标受众画像
   - 创建 Custom Audiences
   - 设置 Lookalike Audiences
   - 选择兴趣定向

3. **账户结构设计**
   - 选择广告系列类型（Promoted Tweets、Promoted Account、Promoted Trends）
   - 组织广告组结构
   - 设置预算分配

4. **出价策略**
   - 选择出价方式：自动出价 / 手动出价 / 最高出价
   - 设置预算上限
   - 配置投放时间

### 2. Creative Optimization

**功能**：优化广告创意以提升点击率和转化率

**执行步骤**：

1. **创意分析**
   - 分析现有创意表现
   - 识别高绩效创意特征
   - 发现低绩效创意问题

2. **创意迭代**
   - 撰写多个广告文案变体
   - 设计不同的视觉素材
   - 测试不同的 CTA

3. **测试设计**
   - 设计 A/B 测试框架
   - 确定测试变量
   - 设置统计显著性要求

### 3. Performance Analysis

**功能**：分析广告表现并提供优化建议

**执行步骤**：

1. **数据收集**
   - 获取广告系列表现数据
   - 收集竞争洞察数据
   - 获取行业基准数据

2. **指标分析**
   - 分析核心指标：Impressions / Clicks / CTR / Conversions / CPA / ROAS
   - 识别表现异常
   - 发现优化机会

3. **归因分析**
   - 多触点归因
   - 首次触点归因
   - 最后触点归因

### 4. Budget Optimization

**功能**：优化广告预算分配

**执行步骤**：

1. **预算分析**
   - 评估当前预算效率
   - 分析各广告系列 ROI
   - 识别预算浪费

2. **重新分配**
   - 建议预算从低效广告系列重新分配
   - 建议增加高效广告系列预算
   - 设置预算优化规则

3. **Scaling**
   - 识别成功扩展机会
   - 设计扩展策略
   - 预防扩展风险

## Operating Guidelines

### Account Health Rules

1. **账户稳定性**
   - 保持稳定的日预算，避免大幅波动
   - 账户评分维护：避免违规内容、保持高质量落地页

2. **投放安全**
   - 遵守 X Ads 政策
   - 避免敏感话题
   - 确保落地页合规

### Optimization Principles

1. **数据驱动**
   - 基于数据决策，不凭直觉
   - 确保统计显著性
   - 记录所有测试结果

2. **渐进优化**
   - 小幅多次调整
   - 每次只改变一个变量
   - 留出足够的测试时间

3. **风险管理**
   - 设置预算上限
   - 保持测试组
   - 及时止损

### Common Patterns

**新广告系列冷启动**：
- 预算设置为预期日预算的 2-3 倍
- 使用自动出价
- 先跑通再优化
- 3-7 天后评估

**表现下降时的优化流程**：
1. 检查账户状态
2. 查看变化曲线
3. 分析受众饱和度
4. 测试新创意
5. 调整出价

**预算扩展策略**：
1. 确认当前 ROAS > 2
2. 逐步增加预算（20-30% 幅度）
3. 监控关键指标
4. 扩展成功后再扩展下一个

## Error Handling

### Common Issues

| 问题 | 可能原因 | 解决方案 |
|---|---|---|
| 广告未获批 | 违规内容/落地页问题 | 修改内容/检查落地页合规性 |
| 花费过慢 | 出价过低/受众过窄 | 提高出价/扩大受众 |
| CPA 过高 | 受众不精准/创意不吸引 | 重新定向/测试新创意 |
| 数据延迟 | X 数据处理延迟 | 等待数据同步 |

### Escalation Protocols

4. 技术故障影响投放

- ✗ 不会绕过 X Ads 政策
